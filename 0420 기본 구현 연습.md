### 2. 다음의 HTTP status code의 의미를 간략하게 작성하시오.

```
200 - 요청이 성공했음
400 - 서버가 요청을 이해 할 수 없음 (bad gate)
401 - 해당 리소스에 유효한 인증 자격 증명
403 - 요청은 전달되었으나, 권한이 없음
404 - 요청한 URL에 해당하는 웹 자원을 찾을 수 없음
500 - 서버의 동작에서 발생하는 에러
```
